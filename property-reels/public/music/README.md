# Background Music for Property Reels

Drop royalty-free music tracks here. They'll be available via `staticFile("music/track-name.mp3")`.

## Recommended Sources (Free for Commercial Use)

- **Pixabay Music** — pixabay.com/music (free, no attribution required)
- **Uppbeat** — uppbeat.io (free tier with attribution)
- **Artlist** — artlist.io (paid, unlimited license)

## Usage

### In Remotion Studio
Set the `musicTrack` prop to the filename:
```
musicTrack: "music/upbeat-corporate.mp3"
```

### In render scripts
Pass `--music=music/chill-mediterranean.mp3` or set it in the theme config.

### Via URL
You can also use a direct URL:
```
musicTrack: "https://example.com/track.mp3"
```

## Recommended Track Style
- **Property Reels (15s):** Upbeat, modern, corporate — think "lifestyle showcase"
- **Carousels (30s):** Chill Mediterranean, acoustic, or light electronic
- Keep tracks under 1MB for fast rendering
- MP3 or WAV format
