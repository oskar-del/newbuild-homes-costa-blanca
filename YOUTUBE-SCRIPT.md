# YouTube Video Listing Script

## How to use
1. Copy the PROMPT below into Claude (browser or app)
2. Paste in the property/area info from your website
3. Claude generates: Title, Description, Tags, Thumbnail text suggestions
4. Copy into YouTube Studio

## QUICK REFERENCE — Optimal YouTube Settings

### Video Settings
- Category: Travel & Events (or Education)
- Language: English
- Recording date: Set to actual filming date
- License: Standard YouTube License
- Allow embedding: YES (important for your website)
- Publish to subscriptions feed: YES

### Cards & End Screens
- Add card at 30% through video → link to website
- Add card at 70% → link to related video
- End screen: Subscribe button + Best related video + Website link
